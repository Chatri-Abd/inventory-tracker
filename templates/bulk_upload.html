📝 CSV Format Guide

Required Column:

  • name - Item name (required)

Optional Columns:

  • description - Item description
  • category - Item category (Tools, Electronics, etc.)
  • location - Where the item is stored
  • quantity - Number of items (defaults to 1)

Example CSV Content:

name,description,category,location,quantity
Screwdriver Set,Phillips and flathead screwdrivers,Tools,Garage,1
Laptop Charger,Dell 65W USB-C charger,Electronics,Office,2
First Aid Kit,Emergency medical supplies,Safety,Car,1
Bluetooth Speaker,Portable wireless speaker,Electronics,Home,1

💡 Tips for Success:

  • ✅ Use UTF-8 encoding for special characters
  • ✅ Each item will get a unique ID automatically (P0000001, P0000002, etc.)
  • ✅ QR codes are generated automatically for each item
  • ✅ Empty cells in optional columns are okay
  • ✅ Quantity defaults to 1 if not specified or invalid
  • ⚠️ Duplicate names are allowed (they'll have different IDs)
